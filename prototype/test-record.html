Superadmin 2.2.2

Test suite (1/5): Backend

Test (1/10): Create new backend

Prerequisities

  1. Running superadmin
  2. User must be logged in
  3. User must be abble to create new item

Steps

  1. Click on New
  2. Click on Backend
  3. Fill in "name" into the field Name
  4. Fill in "version" into the field Version
  5. Fill in "backend" into the field Backend
  6. Fill in "URL" into the field Service URL
  7. Fill in "URL" into the field Documentation location
  8. Click on Save
  9. Verify button Save is not active
